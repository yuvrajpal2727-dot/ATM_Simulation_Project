# ATM_Simulation_Project
# 🏦 ATM Simulation Project

A simple and well-structured **ATM Simulation System** built using Python.
This project demonstrates core programming concepts like **Object-Oriented Programming (OOP)**, modular design, and a **menu-driven interface**.

---

## 🚀 Features

* 💰 Display Account Balance
* ➕ Deposit Money
* ➖ Withdraw Money (with balance validation)
* 📜 Transaction History (Statement)
* 🔁 Continuous menu-driven system using `while True` loop

---

## 🧠 Concepts Used

* Object-Oriented Programming (Classes & Objects)
* Modular Programming (Package Structure)
* Separation of Concerns
* Clean Code Practices

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/atm_project.git
```

2. Navigate to the project folder:

```
cd atm_project
```

3. Run the program:

```
python main.py
```
